== Install

=== Install psycopg2
. Install python-dev nad libpq-dev
+
.Ubuntu 18.04 LTS
[source,console]
----
$ sudo apt install -y python-dev libpq-dev
----

. Install pip packages
+
[source,bash]
----
$ python -m pip install -r requirements.psycopg2.txt
----

=== Install pg8000

. Install pip packages
+
[source,bash]
----
$ python -m pip install -r requirements.pg8000.txt
----
