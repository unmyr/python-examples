== Numpy examples

=== Basic Array
[source,python]
----
>>> import numpy as np
>>> np.array([1, 2, 3])
array([1, 2, 3])
>>> np.array([[1, 2, 3], [4, 5, 6]])
array([[1, 2, 3],
       [4, 5, 6]])
>>>
----

[source,python]
----
>>> v1 = np.array([1, 2, 3])
>>> np.array([v1, [4., 5., 6.]])
array([[1., 2., 3.],
       [4., 5., 6.]])
>>>
----
